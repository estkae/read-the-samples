The popular Excel shortcut keys are Ctrl + C to copy and Ctrl + V to paste, Ctrl + B to bold, Ctrl + I to italicize and Ctrl + U to underline. Ctrl + M activates the Enter key. Pressing the F1 key opens the help file for the active program. The Windows key launches the Start menu and the arrow keys highlight and select items on the screen. For many other common key combinations, check out www.usingtech.com/excel.\n\nWhen you're using Excel's screen controls, it helps to know the shortcut key combinations. In addition to copying, pasting and formatting, the Ctrl key also serves as a modifier key that lets you move the cursor to another area of the screen or to navigate through lists.
